# 🎙️ Sistema de Transmisión P2P HD

Plataforma estática de transmisión de audio punto a punto de baja latencia con códec Opus a 48kHz. Desarrollada para uso de unidades móviles de radio y televisión sin infraestructura de servidores pesada.

## 🚀 Despliegue Express
1. Sube `index.html` y `styles.css` a tu repositorio en GitHub.
2. Ve a las **Settings** (Configuraciones) del repositorio, entra a **Pages**.
3. Selecciona la rama `main` (o `master`) y guarda. ¡Tu enlace estará vivo en menos de un minuto!
